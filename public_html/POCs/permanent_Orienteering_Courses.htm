Permanent Orienteering Courses

Mole Valley manages a network of permanent orienteering courses. Permanent course maps have a selection of controls printed on them, and you can visit as many or as few as you like in one session. 

Permanent orienteering courses are ideal as an introduction to orienteering, or for use as training areas. 

The numbers used to reference the permanent courses are Ordnance Survey grid references. As an alternative follow the links provided to an external web site for a map.

All our maps are available online without charge and can be obtained by completing the form here. You can download an accompanying leaflet below. Should you have a bulk order (50+) then please email us to arrange for them to be printed (£1 per map and leaflet). 

If you have any queries about Permanent Orienteering Courses send an email to the coordinator using the address pocsATmvoc.org (replace the word 'AT' with the symbol '@')

### NEW ### Some of the Permanent Orienteering Courses are now available as virtual courses. Further details can be found here.
