PERMANENT ORIENTEERING COURSES

Mole Valley manages a network of permanent orienteering courses throughout our area. Permanent course maps have a selection of controls printed on them, and you can visit as many or as few as you like in one session. 

Permanent orienteering courses are ideal as an introduction to orienteering, or for use as training areas. 

The numbers used to reference the permanent courses are Ordnance Survey grid references. As an alternative follow the links provided to an external web site for a map.

External maps provided by www.multimap.com

To order any of the permanent course packs direct from Mole Valley, please print the order form and follow its instructions. 

If you have any queries about Permanent Orienteering Courses send an email to the coordinator using the address pocsATmvoc.org (replace the word 'AT' with the symbol '@')
